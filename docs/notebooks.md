# Notebooks

Customer-scoped dashboards that combine metrics and flows. Each notebook belongs to one tenant and can be made public for customer-facing status pages.

## Features

- **Block-based editor** - Drag and arrange metric blocks, flow blocks, and text
- **Drill-down** - Click any visualization to see historical data
- **Public sharing** - Generate URLs for customer-facing status pages
- **Time windows** - Relative or absolute date ranges
- **Annotations** - Mark deployments, incidents, or events on timelines

## Use Cases

- **Support dashboards** - Give your team instant visibility into customer health
- **Status pages** - Let customers self-serve their system status
- **SLA tracking** - Monitor and share performance metrics

## Block Types

**Metric Blocks** - Display metric values

**Flow Status Block** - Show success rate, status breakdown, or duration stats

**Column Layout** - Arrange blocks side-by-side

**Heading** - Add section titles and descriptions

## Filtering

Apply global filters to the entire notebook, or block-level filters for specific metrics. Works with any label from your metrics/flows.
